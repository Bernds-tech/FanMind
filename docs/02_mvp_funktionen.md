# FanMind MVP Funktionen

## Muss-Funktionen für die erste Demo

### 1. Startseite

- klare Erklärung von FanMind
- Zielgruppen: Creator und Fans
- Call-to-Action für Creator
- Call-to-Action für Fans
- seriöser europäischer Plattformcharakter

### 2. Sprachumschaltung

- Deutsch
- Englisch

Zum Start genügt eine statische Umsetzung.

### 3. Creator-Profil

Ein Creator-Profil zeigt:

- Name
- Kurzbeschreibung
- Profilbild-Platzhalter
- Fanclub-Angebot
- Beispielbeiträge
- Mitgliedschaftsoption

### 4. Beiträge

Zum Start als Mockup:

- freier Beitrag
- Mitgliederbeitrag
- Premiumbeitrag

### 5. Dashboard

Das Dashboard zeigt:

- Begrüßung
- Statistiken als Mockup
- neue Fans
- Einnahmen als Mockup
- Beiträge
- Mitgliedschaften

### 6. Admin

Admin-Grundstruktur:

- Nutzerübersicht
- Creator-Übersicht
- Content-Status
- Plattformstatus

## Später notwendig

- echte Registrierung
- Login
- Rollenmodell Fan/Creator/Admin
- Zahlungsanbieter
- Datenbank
- Datei-Upload
- Video-Hosting
- Moderation
- Rechtstexte
- Altersprüfung nur falls später notwendig

## Bewusste Nicht-Funktionen zum Start

- keine komplexen Chats
- keine Livestreams
- keine App
- keine Erotiklogik
- keine Blockchain
- keine Tokenisierung
